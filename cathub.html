HTML: What is it?

HTML (Hyper Text Markup Language) Is a programming language used on the WWW (WorldWideWeb)/ The Internet to display WebSites/Pages. It is written using Tags, Which add meaning to otherwise meaningless texts.

HTML is efficient because back when it was invented was when Internet connections were slow- measuring in the Kilobits per second. HTMLis transmitted as plain text but is interpreted and formatted by the Browser using the tags.

HTML has received many iterations over the years, with the major versions being 1.0, 2.0, 4.0, and 5.0 (3.0 was scrapped). With each version came new Tags and a new era of the Internet!
